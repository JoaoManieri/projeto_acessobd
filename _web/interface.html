Processo seletivo - Tecnomotor


REQUISITOS


1. Leitura e escrita de dados em um banco de dados, seja ele local ou em nuvem;
2. Leitura e escrita de dados em arquivo local (Arquivo de configurações);
3. Separação das camadas de visão e dados utilizando alguma arquitetura de software como, por exemplo: MVC, Microservices,...;
4. O app poderá ser escrito em qualquer linguagem de programação;
5. O aplicativo poderá ser escrito para um dos seguintes sistemas operacionais: Windows ou Android;
6. O código-fonte deverá ser versionado e salvo em seu repositório pessoal (p. ex.: GibHub,GitLab, BitBucket, …);

DESEJAVEL


1. Documente a sua solução como o propósito do seu programa, seu modelo de banco de dados, entre outras documentações que julgar necessário, também no repositório de versionamento;
2. Realize mais de um commit durante a evolução do seu código e comente as alterações decada commit;
3. Inclua os arquivos binários de execução (*.exe, *.jar, *.apk, …) do seu projeto no repositório git com instruções de execução;
4. Envie prints e/ou vídeos do programa em funcionamento: poderá carregar no repositório deversionamento ou salvar um link no repositório que aponta para os arquivos de mídia.
